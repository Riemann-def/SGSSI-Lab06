Instrucciones para el código de SGSSI-Lab06

Descargar:
- Abre Git Bash o Command Prompt.
- Ejecuta: 
   git clone https://github.com/TU_USUARIO/SGSSI-Lab06.git

Ejecutar:
- Navega a los directorios lab5 y lab6 donde se encuentran los archivos .py
   cd ruta_al_directorio\SGSSI-Lab06
- Ejecuta:
   python nombreDeArchivo.py
